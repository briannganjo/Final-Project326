Automobile Data Analysis


File Name: 
'NSFinalProject326.ipynb'


Program Description:
Our program provides an analysis for a set of data that tracks various aspects of automobiles.
It answers the research question of whether there is a correlation between city-mpg of a car and the price of the car.
It provides descriptive statistics, a box plot, a joint plot, a distribution plot, T-Test, and two Regression Analysis.
